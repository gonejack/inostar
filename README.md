# inostar
Command line for converting inoreader starred.json to html

### Install
```shell
> go get github.com/gonejack/inostar
```

### Usage
```shell
> inostar starred.json
```
```
Usage:
  inostar *.json [flags]

Flags:
  -e, --offline   download remote images and replace html <img> references
  -v, --verbose   verbose
  -h, --help      help for inostar
```
